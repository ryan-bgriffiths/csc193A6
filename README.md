# Cryptogram Generator

A simple web-based cryptogram generator that encrypts user input using a shift cipher.

Built using HTML, CSS, and JavaScript as part of a UI and event-handling lab.

## Live Preview
[Click here to try it](Add pages link)

## Features
- Encrypts text using a shift cipher (a → b, z → a)
- Handles both uppercase and lowercase input
- Ignores non-alphabet characters
- Reset button to clear input
- Interactive UI with button events

## How to Run
1. Clone the repository:
   git clone https://github.com/YOUR_USERNAME/cryptogram-generator.git

2. Open the project folder

3. Open `index.html` in your browser

## Example
Input:
hello world

Output:
ifmmp xpsme
